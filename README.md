# MongoDB-API-Node-Express-Starter-Pack

MongoDB API With NodeJS and Express Start Pack 1.0 (with user model and authentication flow in the box). 

## Dependencies

* Mongoose
* Express
* Express Promise Router
* Express handlebars
* Express messages
* Express session
* Helmet
* Joi
* Morgan
* Body-parser
* Connect-flash
* Bcryptjs
* Bootstrap 4

## Installing

```
Yarn install
```
